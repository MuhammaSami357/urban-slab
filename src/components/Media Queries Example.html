Is image mein jo screen dikh rahi hai uska monitor landscape orientation mein hai aur typical desktop view chal raha hai. Agar aap kisi website ya React app ke liye media queries banana chahte hain jisme yeh detect karna ho ke user desktop pe hai ya nahi, to aap ye media queries use kar sakte ho:

1. Desktop Screens (Large Monitors)

css
@media (min-width: 1200px) {
  /* Styles for large desktop screens */
}

2. Standard Desktop/Laptop Screens

css
@media (min-width: 992px) and (max-width: 1199px) {
  /* Styles for normal desktops/laptops */
}

3. Tablet Screens

css
@media (min-width: 768px) and (max-width: 991px) {
  /* Styles for tablets */
}

4. Mobile Screens

css
@media (max-width: 767px) {
  /* Styles for mobile phones */
}

Agar aap chah rahe ho ke layout sirf tab change ho jab screen chhoti ho (jaise mobile/tablet), to aap mostly ye use karo:

css
@media (max-width: 768px) {
  /* Mobile and tablet friendly styles here */
}

Zarurat ho to mujhe batao ke kis element ke liye media query chahiye (navbar, image, grid layout, etc.), main specific example de dunga.

✅ 2. Mobile Queries Display: None Too Much

css
@media (max-width: 767px) {
  .homepage-container {
    display: none;
  }
  .contact-con {
    display: none;
  }
  ...
}

Issue: Tumne bohot saare desktop elements ko mobile pe "display: none" kar diya hai. Yeh thik hai agar unka mobile version separately banaya ho, lekin agar nahi banaya toh layout empty lag sakta hai.

✅ 3. No General Desktop Media Query

Tumne @media (min-width: 1200px) ka structure start kiya hai, but uska content CSS file ke end pe truncate ho gaya hai. Shayad kuch CSS apply hi nahi ho rahi uske andar.

✅ 4. Grid Layouts Mobile Pe Break Nahi Kar Rahe Properly

Tumhare .products, .arrival-products, etc. mein fixed width (330px) lagayi hai, jo mobile pe overflow create karti hai.

Fix Example:

css
@media (max-width: 767px) {
  .pro-one, .pro-two, .pro-three {
    width: 100%; /* Ya 48% for 2-column grid */
  }
}

✅ 5. Scroll ya Zoom Issue ho sakta hai

.product-image mein fixed height 700px hai. Mobile ya tablets pe zoom-out ya overflow hone lagega.

Fix:

css
@media (max-width: 767px) {
  .product-image {
    height: auto;
  }
}

📌 Suggested Media Query Structure (Use This for Clarity):

css
/* Mobile (phones) */
@media (max-width: 767px) { ... }

/* Tablets */
@media (min-width: 768px) and (max-width: 991px) { ... }

/* Laptops */
@media (min-width: 992px) and (max-width: 1199px) { ... }

/* Desktops */
@media (min-width: 1200px) { ... }
